# 🤖 AI Resume Analyzer & Career Guidance Platform

An intelligent web application that analyzes resumes, checks job-role compatibility, and provides career guidance.

Built using **Python and Streamlit**, this tool helps users improve their resumes and understand the skills required for different tech careers.

---

## 🚀 Features

### 📄 AI Resume Analyzer
- Upload your resume (PDF)
- Select a job role
- Get **job match percentage**
- Detect **missing skills**
- Resume **keyword visualization (Word Cloud)**

### 🎯 ATS Resume Score
- Calculates **ATS compatibility**
- Suggests improvements for better ATS ranking

### 🎓 Career Guidance
- Explore **15+ technology domains**
- View required skills for each domain
- Learn about career opportunities
- Salary estimation with **currency conversion**

### 💼 Job Platforms
Direct links to apply for jobs on:

- LinkedIn
- Indeed
- Glassdoor
- Naukri
- AngelList

### 📚 Learning Resources
If you are missing skills, the platform recommends learning platforms like:

- YouTube
- Coursera
- freeCodeCamp
- Kaggle
- W3Schools
- GeeksforGeeks

---

## 🛠 Technologies Used

- Python
- Streamlit
- Natural Language Processing (NLP)
- Matplotlib
- WordCloud

---

## 📊 How It Works

1. Upload your resume
2. Select a job role
3. The system extracts text from your resume
4. Skills are matched with required job skills
5. The system calculates the **match score**
6. Missing skills and suggestions are shown
